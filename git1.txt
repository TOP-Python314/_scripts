УР — удалённый репозиторий
ЛР — локальный репозиторий



=====  удалённый репозиторий  =====

main.py     
funcs.py    шаг 5: получение коммитов из ЛР, обновление данных



шаг 0: clone — копирование файлов и истории изменений из УР в указанный каталог локальной файловой системы и создание ЛР в данном каталоге

=====  локальный репозиторий  =====

main.py     
funcs.py    шаг 1: внесение изменений
            
            шаг 2: add — планирование изменений для ближайшего коммита
            
            шаг 3: commit — фиксация всех запланированных изменений, создание отдельного коммита
            
            шаг 4: push — отправка коммитов из ЛР в УР, синхронизация веток
